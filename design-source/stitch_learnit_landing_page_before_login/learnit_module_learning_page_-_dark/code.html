Understanding Variables in Python

schedule 10 min read
signal_cellular_alt Beginner

Variables are fundamental building blocks in Python. Think of them as containers for storing data values. In Python, a variable is created the moment you first assign a value to it.

Unlike other programming languages like C++ or Java, Python has no command for declaring a variable. This makes it incredibly flexible and easy to start with.

lightbulb

Pro Tip

Variable names are case-sensitive. age, Age, and AGE are three different variables.

Assigning Values

You assign a value to a variable using the equals sign =. The variable name goes on the left, and the value goes on the right.

main.py 
x = 5
y = "John"

# Display the values
print(x)
print(y)

Output will be: 5 and John
